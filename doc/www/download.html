Download GDL

GDL compiles "out of the box" on Linux, Mac OS X (since Panther) and several other UNIX systems (see requirements for details).

The source code distribution is available from the SF.net project summary page.

The latest development version of GDL can be obtained via CVS - consult the SF.net project development page for details. It is also possible to browse the CVS repository using a web-based interface.

There are numerous packaged versions of GDL available for various OSes, including:

  • Linux
    • Fedora
    • Debian
    • Gentoo
    • Arch Linux
    • Ubuntu
  • Mac OS X
    • Macports
    • HPC page
    • HMUG
    • Fink
  • BSD
    • FreeBSD

Please do check the version number of the package you intend to use against the current GDL source-code release. GDL is still in beta stage of development and thus it is highly recommended to use a resonably recent version.

Please note as well that several features of GDL depend on compile-time configuration, and might not be available in pre-built or pre-configured packages.
